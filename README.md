# QuickScore

QuickScore is a simple CGPA calculator designed for university students to calculate their cumulative GPA easily.

## Features

- Add unlimited courses
- Supports Nigerian 5.0 grading system
- Calculates overall CGPA across semesters
- Simple and clean interface
- Mobile friendly

## Tech Stack

- HTML
- CSS
- JavaScript

## How It Works

1. Add your courses
2. Enter the credit unit
3. Select the grade
4. Click calculate

QuickScore will automatically compute your CGPA.

## Grade System

A = 5  
B = 4  
C = 3  
D = 2  
E = 1  
F = 0
